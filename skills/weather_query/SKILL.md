# 天气查询技能

## 功能范围

- 实时天气查询：获取当前温度、湿度、风速、气压等天气数据
- 天气预报查询：获取未来几小时到7天的天气预报
- 空气质量查询：获取PM2.5、PM10、AQI等空气质量指标
- 天气预警：获取恶劣天气预警信息
- 历史天气查询：查询历史天气数据

## 使用规范

1. 当用户询问天气相关问题时使用此技能
2. 需要明确具体的城市名称或地理位置
3. 支持中英文城市名称查询
4. 可以查询当前天气和未来天气预报
5. 可以询问特定气象指标（如温度、湿度、风速等）

## 查询示例

- "北京今天天气怎么样？"
- "上海明天的天气预报"
- "广州的空气质量如何？"
- "深圳未来三天的天气"
- "纽约现在的温度是多少？"

## 部分工具说明

- current_weather: 查询当前实时天气信息
- weather_forecast: 获取未来天气预报
- air_quality: 查询空气质量指数
- weather_alert: 获取天气预警信息
- historical_weather: 查询历史天气数据
